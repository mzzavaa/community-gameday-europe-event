# Contributing

This repository holds the event-specific configuration for AWS Community GameDay Europe. It is the companion to [`community-gameday-europe-stream-templates`](https://github.com/linda-mhmd/community-gameday-europe-stream-templates), which contains all stream visual code.

## Running your own edition

Fork this repo, update the config files, and push. The build and deploy happen automatically via GitHub Actions. See the [README](README.md) for the full setup steps.

## Contributing to this edition

If you spot a mistake in participant data, schedule, or insert prep:

1. Open an issue describing the problem
2. Or open a PR with the fix directly — changes to `config/` and `public/faces/` are straightforward

## Contributing stream template improvements

For changes to compositions, inserts, the web player, or the design system, contribute to the template repo instead:
[`community-gameday-europe-stream-templates`](https://github.com/linda-mhmd/community-gameday-europe-stream-templates)

See its [CONTRIBUTING.md](https://github.com/linda-mhmd/community-gameday-europe-stream-templates/blob/main/CONTRIBUTING.md) for details.
