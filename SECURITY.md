# Security Policy

## Scope

This repository contains only event configuration files (TypeScript config, face photos, schedule data). It has no backend services, no secrets, and no authentication logic.

**In scope:**
- Sensitive data accidentally committed (personal information, tokens)
- Supply chain issues in the build workflow dependencies

**Out of scope:**
- The deployed GitHub Pages site (static read-only)

## Reporting a Vulnerability

Please do **not** open a public issue for security concerns.

Report via [GitHub private security advisories](https://github.com/linda-mhmd/community-gameday-europe-event/security/advisories/new).

We will acknowledge within 48 hours.
